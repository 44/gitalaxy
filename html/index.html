Legend

Visualization of the history of repository. Each star represents a file, the more changes it has, the brighter it shines. Different types of files are represented by different colors:

  • test related
  • localization and logging
  • project files
  • headers
  • everything else

Controls

  • Use mouse to see star details
  • 0 - pause
  • 1 - set speed to day/second
  • 2 - set speed to week/second
  • 3 - set speed to month/second
  • R - Restart from beginning
  • C - Show/hide constallations
  • H / ? - Toggle help
